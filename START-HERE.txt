        Navigate to this folder:
        cd /Users/halim/Documents/singulars-theme-voting

STEP 3: Install Dependencies
        Run this command:
        npm install

================================================================================

TROUBLESHOOTING:

Q: "npm: command not found"
A: You need to install Node.js (see STEP 1)

Q: "Port 3000 already in use"
A: Something else is using port 3000. Either:
   - Close that program, or
   - Use a different port: PORT=3001 npm start

Q: Still getting "Failed to fetch"?
A: Make sure the server is running in the terminal!
   Look for "Server is running on http://localhost:3000"

================================================================================

DATABASE SETUP:
The SQLite database (themes.db) is created automatically when you first
start the server. No manual setup needed!

It's a simple file-based database that stores all themes and votes.

================================================================================

KEEPING IT RUNNING:
The server only runs while the terminal window is open.
To stop it: Press Ctrl+C in the terminal

For production deployment, see README.md
